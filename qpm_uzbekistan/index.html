QPM Model — Structure & Parameters
Uzbekistan Calibration

Core Equations

IS: gap_t = b₁·gap_{t-1} − b₂·MCI_t + b₃·gap*_t + ε_gap
PC: π_t = a₁·π_{t-1} + (1−a₁)·Eπ_{t+1} + a₂·RMC_t + a₄·Δpm_t + ε_π
Δpm_t = Δs_t (quarterly import price change; world price dev=0)
TR: i_t = g₁·i_{t-1} + (1−g₁)·[Eπ_{t+1} + g₂·π4_{t+4} + g₃·gap_t] + ε_i
UIP: s_t = (1−e₁)·Es_{t+1} + e₁·s_{t-1} − (i_t−ρ_t)/4 + ε_s
MCI = b₄·RR_gap − (1−b₄)·Z_gap
RMC = a₃·gap + (1−a₃)·Z_gap
RR_gap = RS − Eπ_{t+1} − RR_BAR
Z_gap = (L_S + L_CPI_RW − L_CPI) − L_Z_BAR
D4L_S = L_S − L_S{−4} (YoY NER)
D4L_CPI= L_CPI − L_CPI{−4} (YoY inflation)
gap*_t = 0.75·gap*_{t-1} + ε_ext (foreign output gap AR(1))

Solver: Bidirectional Gauss-Seidel

The rational expectations model is solved by iterating between:


  • Backward sweep (T→0): UIP exchange rate equation — s[t] depends on s[t+1] (forward-looking). Sweeping backward ensures s[t+1] is already updated in the current pass.
  • Forward sweep (0→T): IS curve, Phillips curve, Taylor rule — depend on lagged values gap[t-1], π[t-1]. Also uses current-period RS in rr_gap (not lagged).
  • RER gap: Properly computed as Z_gap = s[t] − ΣΔπ[k]/4 (CPI level cumulated from QoQ inflation).
  • Convergence: ‖Δπ‖, ‖Δs‖, ‖Δgap‖ < 10⁻¹⁰ (max 600 iterations)

Q0 on all charts = the shock-impact period (same convention as MATLAB/IRIS).

Uzbekistan-Specific Notes

  • a₂ = 0.20 (updated from 0.15) — stronger marginal cost pass-through reflecting rising wage pressures
  • a₃ = 0.65 (updated from 0.60) — higher domestic cost share as structural reforms deepen
  • a₄ = 0.12 — direct import price pass-through (Campa & Goldberg 2005); Uzbekistan imports ≈35% of absorption, so a 10% UZS depreciation → +1.2pp CPI within one quarter via this channel alone
  • e₁ = 0.70 — UZS is largely backward-looking due to limited FX market depth
  • g₁ = 0.80 — CBU rate gradualism
  • π* = 5% — CBU medium-term inflation target
  • r̄ = 3.5% — neutral real rate (structural premium over EM average)
  • Balassa-Samuelson: RER trend appreciation of −1% p.a. from productivity catch-up
  • b₃ = 0.30 (active): external demand transmits via the IS curve; gap*_t follows AR(1) with ρ=0.75
