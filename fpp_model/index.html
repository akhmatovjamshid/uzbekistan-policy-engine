What is the FPP Model?

The Financial Programming and Policies (FPP) framework is an IMF analytical tool that ensures consistency across four macroeconomic sectors: Real, External, Fiscal, and Monetary. Originally developed as the "monetary approach to balance of payments," it was extended to a multi-sector framework by Polak (1957) and refined over decades of IMF practice.

The core insight: any inconsistency between sector-level targets must manifest as a financing gap, reserve drawdown, or monetary accommodation. FPP forces the policymaker to confront these trade-offs explicitly.

This implementation is calibrated to Uzbekistan using IMF FPP data (uzn_fpp_2024_09.xlsx), with base year 2023 and projections to 2026.

Phillips Curve — Inflation Equation

Inflation is projected via a semi-structural Phillips Curve estimated for Uzbekistan:

πt = 1.4 + 0.7·πt−1 + 1.4·y_gapt + 0.28·RER_gapt

Coefficients (from +Inflation sheet):

  • Constant: 1.4 (trend/structural inflation)
  • Persistence (0.70): strong inflation inertia — past inflation strongly predicts current
  • Output gap (1.40): 1pp above potential → +1.4pp inflation
  • RER gap (0.28): exchange rate pass-through; base RER gap = 1.2%

The output gap is computed as the cumulative deviation of actual growth from potential (6.5% p.a.). The RER gap reflects the differential between foreign real rates + risk premium and domestic real rates.

Four-Sector Equations

🔵 Real Sector

Y_nom[t] = Y_nom[t-1] · (1 + g/100) · (1 + π/100) y_gap[t] = y_gap[t-1] + (g - g_pot) [g_pot = 6.5%] π[t] = 1.4 + 0.7·π[t-1] + 1.4·y_gap + 0.28·RER_gap NER[t] = NER[t-1] · (1 + dep/100) Y_USD[t] = Y_nom[t] / NER[t] [bln USD]

🟢 External Sector

CA[t] = (CA_GDP/100) · Y_USD[t] [bln USD] FA[t] = FDI[t] + Other_FA[t] [bln USD] ΔRes[t] = CA[t] + FA[t] Res[t] = Res[t-1] + ΔRes[t]·1000 [mln USD] Cover[t] = Res[t] / (Imports[t]/12) [months] Imports ≈ 35% of GDP (Y_USD·350)

🟡 Fiscal Sector

Revenue[t] = (Rev_GDP/100) · Y_USD[t] Expense[t] = (Exp_GDP/100) · Y_USD[t] Fiscal Bal[t] = Revenue[t] - Expense[t] Primary Bal[t] ≈ Fiscal Bal[t] + 0.3% GDP Financing need = -Fiscal Bal[t] Debt[t] = Debt[t-1] + Financing_need[t]

🟣 Monetary Sector

M2[t] = Y_nom[t] / Velocity[t] [bln UZS] ΔM2[t] = M2[t] - M2[t-1] M2_growth[t] = ΔM2[t] / M2[t-1] · 100 ΔNFA[t] = ΔRes[t] · NER[t] [bln UZS] ΔNDA[t] = ΔM2[t] - ΔNFA[t] [residual] NDA/M2_check: |ΔNDA/M2[t-1]| vs credit plan

Consistency Checks

  • Growth vs Potential: Is real growth near potential (6.5%)? Sustained above-potential growth builds inflationary pressure.
  • Inflation Target: Does Phillips Curve π match the CBU target (8%→6%→5%)? Large deviations signal policy tightening needed.
  • Reserve Adequacy: IMF benchmark = minimum 3 months of imports. Below 3 months signals external vulnerability.
  • Fiscal Sustainability: Deficit > 5% GDP is marked red. Deficit 3-5% GDP is yellow. Below 3% GDP is green.
  • Monetary Consistency: Does ΔNDA implied by money demand match credit growth targets? Large gaps suggest monetary financing of deficit or excessive credit.

Data Sources & Calibration

  • Base year: 2023 (GDP: 1,192,163 bln UZS; 101.2 bln USD)
  • NER (2023): ~11,780 UZS/USD
  • Potential growth: 6.5% p.a. (consensus estimate)
  • RER gap: 1.2% = foreign rate (0.5%) + risk premium (1.5%) − domestic rate (0.8%)
  • Velocity base: 5.62 (2023 estimated)
  • Reserves base: 34,565 mln USD (2023)
  • Import share: ~35% of GDP (BOP data)
  • External debt/GDP: ~38% (2023 estimate)
  • Phillips Curve: Estimated in IMF FPP worksheet for Uzbekistan (2021 vintage)
