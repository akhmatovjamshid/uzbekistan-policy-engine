CAEM Framework — What it is

The CAEM (Country Analysis and Evaluation Model) is the IMF's standardised country-level Financial Programming framework used in Article IV consultations and program negotiations. This implementation is calibrated to Uzbekistan using the CAEM Uzbekistan (Sep 2025) working file.

The framework enforces cross-sector consistency: the savings-investment identity links the current account to fiscal and monetary positions. Any inconsistency between sector targets generates a financing gap that must be identified and resolved.

Base year: 2024 (actual data) | Projection: 2025–2027

Phillips Curve — Inflation (CAEM Sep 2025)

The CAEM calibration uses an open-economy Phillips Curve where imported inflation is the dominant driver (λ₂ = 0.70), reflecting Uzbekistan's high import dependence and exchange rate pass-through:

π_t = 0.70·π_imp + 0.25·π_exp + 0.05·π_{t−1} + 0.40·y_gap

Where: π_imp = NER_dep + π_US (imported inflation) π_exp = 0.5·π_{t−1} + 0.5·π_target (anchored expectations) y_gap = cumulative (g − g_pot), g_pot = 5.8%

The sharp drop in projected inflation (9.6%→3.0% in 2025) is driven by a slowdown in NER depreciation (7.8%→3.3%) and falling US inflation, which sharply reduces imported inflation.

UIP — Exchange Rate

NER depreciation follows an uncovered interest parity (UIP) condition with capital mobility (κ = 0.7):

ΔNER = κ·(r_dom − r_foreign − risk_premium) + (π_dom_target − π_US_target)

2025: dep = 13.5 − 5.4 − 4.1 + (5.0 − 2.0) → adj. = 3.3% 2026: dep = 9.9 − 4.5 − 4.1 + (5.0 − 2.0) → adj. = 5.3%

The UIP model is calibrated with κ=0.7 (partial capital mobility), consistent with Uzbekistan's managed float regime.

Monetary Sector

Money demand follows the quantity equation with velocity as the key link to the real sector:

M2_t = NGDP_t / V_t ΔM2 = ΔNFA_local + ΔNDA_local
ΔNFA_local = Δreserves_USD × NER_avg ΔNDA = ΔM2 − ΔNFA (residual claim)

2024 actuals: M2 = 277 tln UZS; NFA = 362 tln; V = 5.95
Projection note: CAEM projects credit to private sector declining from 38.9% → 22.3% of GDP by 2027, reflecting monetary tightening and UZS appreciation in real terms.

Four-Sector System of Equations

🔵 Real Sector

g_pot = 5.8% (CAEM trend) y_gap[t] = y_gap[t-1] + (g - g_pot) π_imp = NER_dep + π_US π_exp = 0.5·π[t-1] + 0.5·π_target π[t] = 0.70·π_imp + 0.25·π_exp + 0.05·π[t-1] + 0.40·y_gap NGDP[t] = NGDP[t-1]·(1+g/100)·(1+π/100) NER[t] = NER[t-1]·(1+dep/100) GDP_USD[t] = NGDP[t] / NER[t] [mln USD]

🟢 External Sector

CA[t] = (CA_GDP/100)·GDP_USD[t] FDI[t] = (FDI_GDP/100)·GDP_USD[t] Other_FA[t] = (OFA_GDP/100)·GDP_USD[t] ΔRes[t] = CA[t] + FDI[t] + Other_FA[t] Res[t] = Res[t-1] + ΔRes[t] Imports[t] ≈ 35.5%·GDP_USD[t] (avg. share) Cover[t] = Res[t] / (Imports[t]/12) Ext.debt: from financial account flows

🟡 Fiscal Sector

Rev[t] = (rev_GDP/100)·GDP_USD[t] Exp[t] = (exp_GDP/100)·GDP_USD[t] FB[t] = Rev[t] − Exp[t] PB[t] = FB[t] + interest_payments interest ≈ eff.rate × debt_stock Debt[t] = Debt[t-1] + (−FB[t]) Debt_GDP[t] = Debt[t] / GDP_USD[t] × 100 Fin.need = exp_GDP − rev_GDP (% GDP)

🟣 Monetary Sector

M2[t] = NGDP[t] / V[t] [bln UZS] ΔM2[t] = M2[t] − M2[t-1] M2_gr[t] = ΔM2 / M2[t-1] × 100 ΔNFA_uzs = ΔRes[t]·NER_avg[t] / 1000 ΔNDA_uzs = ΔM2 − ΔNFA_uzs (residual) NFA/GDP = NFA_uzs / NGDP × 100 NDA/GDP = NDA_uzs / NGDP × 100 Real policy rate = policy_rate − π

Consistency Checks Explained

  • Growth vs Potential (5.8%): Output gap accumulation. Sustained above-potential growth builds inflation pressure.
  • Inflation vs CBU Target (5%): Phillips Curve projection vs 5% medium-term target. The CAEM baseline shows inflation falling sharply to 3% in 2025 due to NER stabilisation.
  • Reserve Adequacy: IMF ARA benchmark minimum 3 months. CAEM projects 11–12 months — well above threshold.
  • Fiscal Position: CAEM projects fiscal deficit widening to 4.9% in 2025 due to investment expenditure. Green <3%, Yellow 3–5%, Red >5%.
  • Debt Dynamics: CAEM baseline: govt debt rises from 32.6% (2024) → 38.9% (2027) due to increased domestic borrowing. Red >50%, Yellow 40–50%, Green <40%.
