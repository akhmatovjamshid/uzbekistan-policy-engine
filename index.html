QPM — Quarterly Projection Model

The QPM is a calibrated New-Keynesian small open-economy model solved with the IRIS Toolbox in MATLAB. It features four core equations: IS curve (aggregate demand), hybrid New-Keynesian Phillips curve, forward-looking Taylor rule, and a modified UIP condition for the exchange rate. The model is solved using Gauss-Seidel iteration with bidirectional sweeps to handle both backward-looking (UIP) and forward-looking (IS, Phillips, Taylor) dynamics.

Key calibrated parameters: b₁ = 0.70 (output gap persistence), a₁ = 0.60 (inflation inertia), a₂ = 0.20 (marginal cost pass-through), g₂ = 1.50 (Taylor rule inflation response). Steady-state: inflation target 5%, potential growth 6%, neutral real rate 3.5%.

DFM — Dynamic Factor Model

The DFM follows Bańbura & Modugno (2014) and extracts a single latent common factor from 34 mixed-frequency indicators (monthly + quarterly GDP) using the EM algorithm with a Kalman filter/smoother. The factor captures broad economic momentum and is used to nowcast quarterly GDP up to 3 months ahead. Parameters are estimated in R and exported as a JSON file; the prediction step (Kalman filter update) runs live in the browser.
